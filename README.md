# Petify

Petify is an innovative online platform that connects pet owners with certified veterinarians through live chat and video consultations. The platform offers a variety of features that make pet care more accessible and convenient, including a blog with articles, a symptom checker, and an adoption platform.

## Key Features

- **Live Chat with Vets**: Pet owners can chat in real-time with certified vets for immediate advice.
- **Video Consultations**: Schedule video appointments with vets for more in-depth discussions about your pet's health.
- **Symptom Checker**: Use the symptom checker when chat and video services are unavailable to get guidance on your pet's condition.
- **Blog and Articles**: Browse articles written by veterinary experts on pet health, nutrition, and care.
- **Pet Adoption Platform**: Find pets available for adoption or post your pets looking for new homes.
- **User-Friendly Interface**: Designed with simplicity in mind for both pet owners and veterinarians.

## Tech Stack

- **Frontend**: React.js
- **Backend**: Node.js with Express
- **Database**: MongoDB (Atlas)
- **Authentication**: JWT-based system with bcrypt for password hashing
- **Image Storage**: Firebase

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/yourusername/petify.git
    ```

2. Navigate to the project directory:

    ```bash
    cd petify
    ```

3. Install the dependencies:

    ```bash
    npm install
    ```

4. Create a `.env` file in the root directory with the following environment variables:

    ```env
    MONGO_URI=your_mongo_db_connection_string
    JWT_SECRET=your_jwt_secret
    CLOUDINARY_URL=your_cloudinary_url
    ```

5. Start the development server:

    ```bash
    npm run dev
    ```

## Usage

1. **Live Chat**: Connect with a veterinarian instantly via the chat feature.
2. **Video Consultation**: Schedule a video call for more comprehensive consultations.
3. **Symptom Checker**: Enter symptoms to get possible causes and next steps when vets are not available.
4. **Blog**: Read articles to learn more about caring for your pet.
5. **Adopt a Pet**: Browse pets available for adoption or post a pet needing a new home.
